1.2 はじまり

~/progs の作成 [/]

  • [ ] 講義用のディレクトリを作成します： 
# [~/]
cd ~/
mkdir -p progs
cd progs
mkdir -p lects cutter card_display game/{card,deck,hand,poker}
ls -R
  • [ ] ~/progs/README.org を作成します: 
#+title: ~/progs/README.org
*ディレクトリの使用目的:
 - ./lects: 毎回の講義のめもを置きましょう
 - ./card_display: カード表示問題の開発用
 - ./cutter: テスト用
 - ./game: ポーカーゲーム開発用
   - ./game/card: カードモジュール
   - ./game/deck: 山モジュール
   - ./game/hand: 手札モジュール
   - ./game/poker: ポーカーモジュール
